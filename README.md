## Olio란 무엇인가요?
올리오는 소프트웨어마이스터고등학교에서 학생들의 포트폴리오를 책 형식으로 만드는 '드림북'을 대체하기 위한 웹&앱 서비스입니다
<br/><br/>


## 어떤 기능을 제공하나요?

### 포트폴리오

- 포트폴리오 작성
- 포트폴리오 열람
- 포트폴리오 수정

### 프로젝트

- 프로젝트 소개 페이지 작성
- 프로젝트 소개 페이지 열람
- 프로젝트 소개 페이지 수정

### 유저

- 로그인&회원가입
- 아이디&비밀번호 찾기
- 유저 프로필 작성
- 유저 프로필 열람
- 유저 프로필 삭제
- 유저 팔로우
- 유저 좋아요

### 검색

- 포트폴리오 검색
- 프로젝트 검색
- 유저 검색
